# Agent · 智能体演示（Ch.11）

> 在 **Coze.cn** 上现场建一个"月球营报名顾问 Agent"，让它接 3 个工具自动完成"帮我准备 7.15 上月球"这件事。展示 ReAct 循环：思考 → 调工具 → 看结果 → 再思考。

## 🎯 学员看到的体验

1. 你打开 Coze.cn，进到提前建好的「月球营报名顾问」bot
2. 学员问："帮我看看 7 月 15 日大连飞月球，要准备什么？"
3. Agent 自动做这几件事（学员实时看到）：
   - 调"地球天气工具"查大连 7.14-15 天气（出发前一天会不会下雨影响发射）
   - 调"月球时间工具"算地球时刻 ↔ 月球时刻
   - 调"行李清单工具"生成基于天气 + 月球生保系统的清单
4. Agent 综合输出："出发前一天大连小雨 22°C，建议你..."

**与 Chatbot 对比**：Chatbot 只会告诉你"你可以打开携程查..."；Agent 真的去查 + 整合。

## 🎬 现场操作步骤（6 分钟）

### 课前准备

1. Coze.cn 注册账号
2. 按 `coze-bot-setup.md` 步骤建一个 Bot：
   - 名字：「月球营报名顾问」
   - 系统提示：见 setup 文件
   - 添加 3 个工具（天气 / 时间 / 行李清单）
3. 把分享链接生成并贴在 admin 端「演示控制台」备用
4. 课前测试：在 Bot 里输入测试请求，确保 3 个工具都能正常调用

### 翻页到 `#/13/1` (Agent 互动页)

| Step | 动作 | 学员看到 |
|---|---|---|
| 1 | 切到 Coze.cn 你的 Bot | 整个 Bot 界面，标题"月球营报名顾问" |
| 2 | 现场输入 P1（见 `test-prompts.md`） | Agent 开始思考 + 显示工具调用 |
| 3 | 指着工具调用日志，慢慢念 "它在调天气 API"、"它在算时间"、"它在生成清单" | 学员看到 ReAct 循环 |
| 4 | 等 Agent 综合输出 | 一个完整、个性化的回答 |
| 5 | (可选) 输入 P2，让学员看 Agent 处理另一种场景 | — |
