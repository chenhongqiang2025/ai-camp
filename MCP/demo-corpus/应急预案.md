# 月球营·应急预案 v3.2

## A 类·太阳风暴

| 级别 | 触发条件 | 响应 |
|---|---|---|
| 黄 | 太阳风速度 > 600 km/s | 30 分钟内全员归舱顶屏蔽层 |
| 橙 | 太阳风速度 > 800 km/s + 质子通量 > 100 pfu | 立即转入地下 12m 避难舱 |
| 红 | 太阳风速度 > 1200 km/s + X 级耀斑 | 转入深层避难（48h 给养） + 暂停 EVA 至少 72 小时 |

## B 类·月震

- 1–3 级：训练系统自动暂停 30 秒后恢复
- 4–6 级：强制全员归舱 + 自动检查"广寒宫-3"结构完整性
- ≥ 7 级：触发深层避难，等待地月救援评估

## C 类·EVA 紧急召回

触发条件之一：
1. 任意学员"星辰-9"外骨骼生命体征异常（心率 < 50 或 > 150）
2. 月尘风暴预警（基地外能见度 < 50m）
3. "鹊桥-8"中继星通信中断 > 3 分钟
4. 玉兔-10 失控（视觉 SLAM 不收敛 + 主动靠近学员）

召回流程：
1. 基地广播 + 外骨骼震动提醒
2. 学员立即返回最近气闸室（标识：H1/H2/H3）
3. "尘洗"流程 → 监控室确认 → 解除警报

## D 类·医疗

- 现场医师：1 名（基地常驻），来自第九研究院航天医学组
- 远程医疗：通过"鹊桥-8"，可对接地球北京协和医院
- 学员意外险：5000 万人民币起赔（由园区财政承担）
