## 项目简介

由 IBM Consulting · FutureNow Center China 与大连高新园区联合举办的 AI 夏令营，
面向高校学生的生成式 AI 启蒙与互动教学课件。

讲师：陳宏强 Richard Chen（IBM 生成式 AI 解决方案架构师）
开营：2026 年 7 月 1 日

## 内容主线

```
AI 发展简史 → 机器学习与神经网络 → NLP 演进 → ChatGPT 时刻
            ↓
        大模型的幻觉问题
            ↓
RAG · 检索增强生成 → 本体论与知识图谱 → 力导图与关系推理
            ↓
   MCP → Agent → A2A → IBM 的 AI 实践
```

每个核心章节均设计了学生互动环节，扫码即可参与。
