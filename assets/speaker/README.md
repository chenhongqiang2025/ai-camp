# 讲师照片放置规范

把 Richard 的职业头像照命名为 **`richard.jpg`**，放在这个目录下：

```
ai-camp/assets/speaker/richard.jpg
```

## 要求

| 项 | 值 |
|---|---|
| 文件名 | `richard.jpg`（小写，必须） |
| 比例 | 竖版 4:5 或 3:4 最佳 |
| 最小尺寸 | 600 × 800 px |
| 格式 | JPG（PNG 也可，但需要保持文件名 `richard.jpg`，浏览器会按内容识别） |

## 行为

- 图片**不存在时**：讲师介绍页布局保持原样（只有 ID 文字块），不会破图
- 图片**存在时**：讲师介绍页自动变成「左照片 + 右 ID 块」的双栏
- 图片**改了重新 push**：GitHub Pages 自动更新

## 不要

- 不要带任何文字水印
- 不要超过 2 MB（影响加载速度）
- 不要其他名字，必须 `richard.jpg`
